World Percentage Rural Population Data Analysis 2017

A brief visual analysis of ten countries rural population as a percentage of their total population

Overview

The data was taken from World Bank OrganizationTen countries from assignment one were used and the year 2017 was chosen.
The following are representation of the data in the form of a D3 bar chart complete with axes, axes labels,tick marks, tick mark labels, legend and a title. The chart would have the following to animation features.

  • User input to filter the data (show all countries, filter sort by value to show the top 5 countries, show the bottom 5 countries)
  • Reorder the data (sort countries alphabetically (both default start state and to reset the chart), sort by value in ascending order, sort by value in descending order)
The aim is to use D3 to visualize data and to understand how the mentioned countries stack up to each other in terms of rural population. Also the page is being hosted on a live server for easier access.
